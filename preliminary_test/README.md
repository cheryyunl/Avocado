# Preliminary Test

This repository contains the code for the preliminary test.

The `LLaVA-med` model is fine-tuned based on `LLaVA-mistral`.

## Running the Code

To run the code for `LLaVA-Med`, you need to place the code into the original `LLaVA-Med` repository under:
[https://github.com/microsoft/LLaVA-Med.git](https://github.com/microsoft/LLaVA-Med.git)

Specifically, place the code under the following path in the repository:
LLaVA-Med/llava/eval
